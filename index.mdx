## Bienvenido a VIVLA

VIVLA es una plataforma de **co-ownership inmobiliario**. Permite a copropietarios reservar estancias, comunicarse entre sí, gestionar sus propiedades y acceder a servicios de atención al cliente — todo desde una app móvil y un conjunto de herramientas internas.

Esta wiki es la **fuente de verdad** para todo el equipo: ingeniería, producto, operaciones y CX. Aquí encontrarás cómo funciona cada sistema, cómo se conectan entre sí y cómo contribuir.

## Cómo se conecta todo

La app móvil consume **dos backends**: la API principal (booking, auth, propiedades) y la Tools API (chat, encuestas). El panel de CX usa solo la API principal. Vivla Tools sirve además portales públicos para clientes y propietarios.

## Ecosistema completo

VIVLA está compuesto por **11 repositorios** organizados en 4 dominios:

| Dominio | Repos | Descripción |
|---------|-------|-------------|
| **Plataforma principal** | `vivla-api`, `vivla-mobile-app`, `vivla-tools`, `vivla-panel` | Los 4 productos core del negocio |
| **Servicios** | `booking-cloud-functions`, `calendar-manager`, `VivlaNPS` | Automatizaciones, integraciones y herramientas auxiliares |
| **Deep links** | `vivla-app-site`, `app-invites` | Redirects, verificación de links y landing pages |
| **Externo** | `propcos` | Desarrollo por contratista (Digio) |
