## Bienvenido

VIVLA es una plataforma de real estate y booking. Esta documentación cubre la arquitectura, APIs, aplicación móvil, herramientas internas e infraestructura del proyecto.

## Repositorios

| Repo | Stack | Descripción |
|------|-------|-------------|
| `vivla-api` | NestJS + PostgreSQL | API principal (mobile + panel) |
| `vivla-mobile-app` | React Native + Expo | App iOS/Android para copropietarios |
| `vivla-tools` | TanStack Start + NestJS + Supabase | Chat, community, surveys, notificaciones, tools API |
| `vivla-panel` | TBD | Backoffice para agentes CX |
| `booking-cloud-functions` | Firebase Functions | Integraciones: HubSpot, Notion, Slack, Zendesk |
| `calendar-manager` | TBD | Gestión de calendarios |
| `vivla-app-site` | Cloudflare Pages | Deep links y landing de la app |
| `app-invites` | Vercel Functions | Redirects de invitación |
| `VivlaNPS` | React + Firebase | Subida imágenes NPS |
| `propcos` | Externo (Digio) | Desarrollo por contratista |
| `docs` | Mintlify | Este repositorio de documentación |
