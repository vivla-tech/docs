## Bienvenido

VIVLA es una plataforma de real estate y booking. Esta documentación cubre la arquitectura, APIs, aplicación móvil, herramientas internas e infraestructura del proyecto.

## Repositorios

| Repo | Stack | Descripción |
|------|-------|-------------|
| `vivla-backend` | NestJS + PostgreSQL | API principal (mobile + panel) |
| `vivla-mobile` | React Native + Expo | Aplicación móvil iOS/Android |
| `vivla-tools` | Monorepo (Next.js + NestJS) | Chat, community guidelines, guías, tools API |
