## Overview

El chat de soporte permite comunicación en tiempo real entre el equipo de VIVLA y los usuarios. Está disponible tanto en el backoffice de tools como integrado en la app mobile.

## Puntos de acceso

| Plataforma | Ubicación | Rol |
|------------|-----------|-----|
| Tools (web) | Panel de soporte | Agente / Admin |
| Mobile app | Chat en la app | Usuario / Cliente |
