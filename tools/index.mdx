## Overview

`vivla-tools` es un monorepo que contiene las herramientas internas de VIVLA: chat de soporte, community guidelines, guías de mantenimiento, y el backend que las sirve.

## Estructura del monorepo

```
vivla-tools/
├── apps/
│   ├── tools/           # Frontend: mini apps (chat, guías, etc.)
│   └── community/       # Frontend: portal community guidelines (clientes)
└── packages/
    └── backend/         # API NestJS que sirve a todos los frontales
```
