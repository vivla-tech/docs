# Vivla Tools

Monorepo que contiene las herramientas internas de Vivla: un conjunto de aplicaciones para gestionar comunicaciГіn con huГ©spedes, propiedades, guГӯas de mantenimiento, notificaciones y operaciones del equipo.

## Estructura del monorepo

<CardGroup cols={2}>
  <Card title="Frontend" icon="desktop">
    **`apps/frontend`** вҖ” Panel interno principal construido con TanStack Start (React). Incluye los mГіdulos de Chat, Community, Notificaciones, Teams y Dashboard.

    Puerto: `3000`
  </Card>
  <Card title="Backend" icon="server">
    **`apps/backend`** вҖ” API NestJS que sirve a todos los frontales y a la app mobile (vivla-mobile). Gestiona autenticaciГіn, permisos, chat, surveys, community, notificaciones e inbox.

    Puerto: `3001`
  </Card>
  <Card title="Community Frontend" icon="globe">
    **`apps/community-frontend`** вҖ” Portal pГәblico Next.js 14 para que huГ©spedes y personal de limpieza accedan a guГӯas de propiedades, inventario, planos y sesiones de limpieza.

    Puerto: `3002`
  </Card>
  <Card title="Home Excellence" icon="chart-line">
    **`apps/home-excellence`** вҖ” Portal pГәblico Next.js 14 para propietarios: mГ©tricas de satisfacciГіn NPS, estado de calidad, tickets, protocolos y resultados del Home Review 2026.

    Puerto: `3003`
  </Card>
  <Card title="Packages compartidos" icon="cube">
    **`packages/common`** вҖ” Tipos, utilidades, constantes y validaciones compartidas (Zod).

    **`packages/ui`** вҖ” LibrerГӯa de componentes React con shadcn/ui y Tailwind CSS.

    **`packages/eslint-config`** вҖ” ConfiguraciГіn ESLint compartida.
  </Card>
</CardGroup>

## MГіdulos principales

| MГіdulo | Frontend | Backend | DescripciГіn |
|--------|----------|---------|-------------|
| **Chat** | `/app/chat/` | `/api/chat/` | MensajerГӯa en tiempo real con Stream Chat, tickets de soporte, gestiГіn de canales y agentes |
| **Surveys** | `/app/chat/surveys/` | `/api/surveys/`, `/api/survey-types/`, `/api/survey-legacy-responses/`, `/api/survey-scores/`, `/api/chat/sync/firebase-legacy-responses` | Encuestas dinГЎmicas con builder visual, versionado, 5 tipos (Casa, Estancia, Llegada, Onboarding, Financiero), resultados con sub-tabs, user matrix multi-source, rewards, action plans y sync automГЎtico desde Firebase via Windmill |
| **Community** | `/app/community/` | `/api/community/` | GestiГіn de propiedades: guГӯas, inventario, limpieza, planos, fotos, styling |
| **Notificaciones** | `/app/notifications/` | `/api/notifications/` | Push notifications con templates, automaciones, deep links y analytics |
| **Teams** | `/app/teams/` | `/api/teams/` | GestiГіn de equipos, miembros y roles |
| **Inbox** | Sidebar | `/api/inbox/` | Bandeja de entrada unificada con notificaciones en tiempo real |
| **Permisos** | Integrado | `/api/permissions/` | Control de acceso granular por mГіdulo (none/view/edit/admin) |
| **Home Excellence** | `apps/home-excellence/` | `/api/surveys/public/*` | Portal propietarios: NPS, satisfacciГіn, tickets, Home Review 2026 |

## Stack tecnolГіgico

| Capa | TecnologГӯa |
|------|------------|
| Frontend | TanStack Start, TanStack Router, TanStack Query, React 18, Tailwind CSS |
| Community Frontend | Next.js 14 (App Router), React PDF |
| Backend | NestJS 10, Express |
| Base de datos | Supabase (PostgreSQL con RLS) |
| AutenticaciГіn | Auth0 (JWT/JWKS), Stream Chat tokens para mobile |
| Chat en tiempo real | Stream Chat SDK |
| Push notifications | Firebase Admin SDK, Expo Server SDK, Web Push (VAPID) |
| Almacenamiento | Cloudinary (imГЎgenes) |
| Email | Resend |
| Build | Turborepo, pnpm workspaces |
