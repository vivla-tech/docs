# Vivla Tools

Monorepo que contiene las herramientas internas de Vivla: chat de soporte en tiempo real, encuestas NPS, community guidelines, notificaciones push, gestión de equipos y permisos.

## Repositorio

- **Repo:** [vivla-tech/vivla-tools](https://github.com/vivla-tech/vivla-tools)
- **Stack:** TanStack Start (frontend) + NestJS (backend) + Supabase (PostgreSQL)
- **Hosting:** Railway

## Módulos principales

| Módulo | Descripción |
|--------|-------------|
| **Chat** | Mensajería en tiempo real con Stream Chat, tickets de soporte, gestión de canales y agentes |
| **Surveys** | Encuestas dinámicas con builder visual, 5 tipos (Casa, Estancia, Llegada, Onboarding, Financiero) |
| **Community** | Guías de propiedades, inventario, limpieza, planos |
| **Notificaciones** | Push notifications con templates, automaciones y deep links |
| **Teams** | Gestión de equipos, miembros y roles |
| **Permisos** | Control de acceso granular por módulo |
