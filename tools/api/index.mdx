## Overview

La Tools API es un backend NestJS que vive en `packages/backend/` dentro del monorepo vivla-tools. Sirve a:

- **Tools frontend** — panel de soporte, guías de mantenimiento
- **Community portal** — portal de clientes
- **Mobile app** — chat de soporte integrado en vivla-mobile

## Módulos

| Módulo | Descripción | Consumido por |
|--------|-------------|---------------|
| Chat | Mensajería en tiempo real | Tools, Mobile |
| Community | Guías de comunidad | Community portal |
| Maintenance | Guías de mantenimiento | Tools |
