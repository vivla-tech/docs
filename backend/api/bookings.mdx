## Overview

El módulo de bookings gestiona reservas de propiedades, incluyendo disponibilidad, creación y cancelación.

## Endpoints

| Método | Ruta | Descripción | Auth |
|--------|------|-------------|------|
| GET | `/bookings` | Listar reservas del usuario | Sí |
| GET | `/bookings/:id` | Detalle de reserva | Sí |
| POST | `/bookings` | Crear reserva | Sí |
| PATCH | `/bookings/:id/cancel` | Cancelar reserva | Sí |
| GET | `/properties/:id/availability` | Consultar disponibilidad | No |

<Note>
  Completar con los estados de booking, reglas de cancelación y response schemas.
</Note>
