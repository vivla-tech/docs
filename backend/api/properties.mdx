## Overview

El módulo de propiedades gestiona el CRUD de listings inmobiliarios.

## Endpoints

| Método | Ruta | Descripción | Auth |
|--------|------|-------------|------|
| GET | `/properties` | Listar propiedades | No |
| GET | `/properties/:id` | Detalle de propiedad | No |
| POST | `/properties` | Crear propiedad | Sí |
| PATCH | `/properties/:id` | Actualizar propiedad | Sí |
| DELETE | `/properties/:id` | Eliminar propiedad | Sí |

<Note>
  Completar con los query params de filtrado, paginación y response schemas cuando la API esté estable.
</Note>
