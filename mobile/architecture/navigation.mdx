## Overview

VIVLA Mobile usa Expo Router (file-based routing) para la navegación.

## Estructura de rutas

```
app/
├── (tabs)/            # Tab navigator principal
│   ├── index.tsx      # Home / Explorar
│   ├── search.tsx     # Búsqueda
│   ├── bookings.tsx   # Mis reservas
│   └── profile.tsx    # Perfil
├── property/
│   └── [id].tsx       # Detalle de propiedad
├── booking/
│   └── [id].tsx       # Detalle de reserva
├── auth/
│   ├── login.tsx      # Login
│   └── register.tsx   # Registro
└── _layout.tsx        # Layout raíz
```

<Note>
  Actualizar esta estructura cuando las rutas estén definidas.
</Note>
