# Mobile App

App móvil para copropietarios de VIVLA. Permite reservar estancias, comunicarse, gestionar propiedades y perfil desde iOS y Android.

  La documentación completa de la Mobile App se genera automáticamente desde el repositorio `vivla-mobile-app`. Si esta página aparece sin contenido detallado, la agregación aún no se ha ejecutado.
</Note>

## Repositorio

- **Repo:** [vivla-tech/vivla-mobile-app](https://github.com/vivla-tech/vivla-mobile-app)
- **Stack:** React Native 0.79, Expo SDK 53, TypeScript 5.8
- **Hosting:** App Store (iOS), Google Play (Android)

## Módulos principales

| Módulo | Descripción |
|--------|-------------|
| **Auth** | Login, registro, reset de contraseña, modo visitante |
| **Booking** | Reservas de estancias con calendario |
| **Chat** | Mensajería en tiempo real (via Tools API) |
| **Property** | Gestión y detalle de propiedades |
| **Stays** | Estancias actuales y pasadas |
| **Surveys** | Encuestas NPS y de satisfacción |
| **Profile** | Perfil de usuario y configuración |
| **Notifications** | Notificaciones push y en app |
